# 产品客服培训机器人 - 需求文档

## 一、项目概述

### 1.1 项目目标
打造一个基于大语言模型的产品客服培训系统，帮助新客服快速熟悉产品知识、掌握服务话术、提升服务能力。

### 1.2 目标用户
- 新入职的客服人员
- 需要学习新产品知识的老客服
- 需要进行服务能力测评的管理者

### 3.2 API 调用实现要点

**文件：src/api_client.py**

```python
import os
from anthropic import Anthropic
from typing import List, Dict

class ClaudeAPIClient:
    def __init__(self):
        # 从环境变量读取配置（与 settings.json 中的 env 对应）
        self.base_url = os.getenv("ANTHROPIC_BASE_URL")
        self.api_key = os.getenv("ANTHROPIC_AUTH_TOKEN")
        self.timeout = int(os.getenv("API_TIMEOUT_MS", "300000"))
        
        # 初始化客户端
        self.client = Anthropic(
            api_key=self.api_key,
            base_url=self.base_url,
            timeout=self.timeout
        )
        self.model = "claude-opus-4-8-1m"  # 对应 settings.json 中的 opus[1m]
    
    def chat(self, messages: List[Dict], system_prompt: str) -> str:
        """
        发送对话请求
        messages: [{"role": "user"|"assistant", "content": "..."}]
        """
        response = self.client.messages.create(
            model=self.model,
            max_tokens=4096,
            system=system_prompt,
            messages=messages
        )
        return response.content[0].text
```

**关键点：**
- 环境变量从运行时环境读取（ Claude Code 会自动注入 settings.json 中的 env）
- 使用官方 anthropic SDK
- 超时设置与配置保持一致

### 3.3 知识库管理实现要点

**文件：src/knowledge.py**

```python
import os
from pathlib import Path
from typing import List, Dict
import markdown

class KnowledgeManager:
    def __init__(self, knowledge_dir: str = "knowledge"):
        self.knowledge_dir = Path(knowledge_dir)
        self.knowledge_cache = {}
        self.load_all_knowledge()
    
    def load_all_knowledge(self):
        """加载目录下所有 Markdown 文件"""
        if not self.knowledge_dir.exists():
            return
        
        for md_file in self.knowledge_dir.glob("*.md"):
            with open(md_file, "r", encoding="utf-8") as f:
                self.knowledge_cache[md_file.name] = f.read()
    
    def get_knowledge_content(self, filename: str = None) -> str:
        """获取知识库内容，合并或单个"""
        if filename and filename in self.knowledge_cache:
            return self.knowledge_cache[filename]
        return "\n\n".join(self.knowledge_cache.values())
    
    def to_html(self, content: str) -> str:
        """Markdown 转 HTML 用于显示"""
        return markdown.markdown(content)
```

**关键点：**
- 缓存机制避免重复读取文件
- 支持多文件合并
- Markdown 渲染支持

### 3.5 对话管理实现要点

**文件：src/chat_manager.py**

```python
from typing import List, Dict
from dataclasses import dataclass
from datetime import datetime
import uuid

@dataclass
class Message:
    role: str  # "user" | "assistant" | "system"
    content: str
    timestamp: str = None

class ChatSession:
    def __init__(self, session_id: str = None):
        self.session_id = session_id or str(uuid.uuid4())
        self.created_at = datetime.now().isoformat()
        self.messages: List[Message] = []
        self.mode = "free_chat"  # free_chat | role_play | scenario
    
    def add_message(self, role: str, content: str):
        self.messages.append(Message(
            role=role,
            content=content,
            timestamp=datetime.now().isoformat()
        ))
    
    def get_history(self, limit: int = 20) -> List[Dict]:
        """获取对话历史，限制条数避免上下文过长"""
        messages = self.messages[-limit:]
        return [{"role": m.role, "content": m.content} for m in messages]
    
    def to_dict(self) -> Dict:
        return {
            "session_id": self.session_id,
            "created_at": self.created_at,
            "messages": [
                {"role": m.role, "content": m.content, "timestamp": m.timestamp}
                for m in self.messages
            ],
            "mode": self.mode
        }

class ChatManager:
    def __init__(self):
        self.sessions: Dict[str, ChatSession] = {}
    
    def create_session(self) -> ChatSession:
        session = ChatSession()
        self.sessions[session.session_id] = session
        return session
    
    def get_session(self, session_id: str) -> ChatSession:
        return self.sessions.get(session_id)
```

**关键点：**
- 会话隔离，支持多用户
- 历史记录限制，控制 token 消耗
- 数据序列化支持

### 3.6 FastAPI 后端实现要点

**文件：src/main.py**

```python
from fastapi import FastAPI, Request, HTTPException
from fastapi.staticfiles import StaticFiles
from fastapi.templating import Jinja2Templates
from fastapi.responses import HTMLResponse, JSONResponse
from pathlib import Path
import sys

# 导入本地模块
from api_client import ClaudeAPIClient
from knowledge import KnowledgeManager
from chat_manager import ChatManager, ChatSession
from prompts import PromptTemplates

app = FastAPI(title="客服培训机器人")

# 初始化管理器
api_client = ClaudeAPIClient()
knowledge_manager = KnowledgeManager()
chat_manager = ChatManager()

# 模板和静态文件
templates = Jinja2Templates(directory="src/templates")
app.mount("/static", StaticFiles(directory="src/static"), name="static")

@app.get("/", response_class=HTMLResponse)
async def index(request: Request):
    """主页面"""
    return templates.TemplateResponse("index.html", {"request": request})

@app.post("/api/session")
async def create_session():
    """创建新会话"""
    session = chat_manager.create_session()
    return {"session_id": session.session_id}

@app.post("/api/chat")
async def chat(request: Request):
    """发送消息"""
    data = await request.json()
    session_id = data.get("session_id")
    user_message = data.get("message", "")
    mode = data.get("mode", "free_chat")
    
    session = chat_manager.get_session(session_id)
    if not session:
        raise HTTPException(status_code=404, detail="会话不存在")
    
    # 添加用户消息
    session.add_message("user", user_message)
    
    # 获取知识库
    knowledge = knowledge_manager.get_knowledge_content()
    
    # 构建 prompt
    system_prompt = PromptTemplates.get_system_prompt(knowledge)
    
    # 调用 API
    try:
        response = api_client.chat(
            messages=session.get_history(),
            system_prompt=system_prompt
        )
        session.add_message("assistant", response)
        return {"response": response, "session_id": session_id}
    except Exception as e:
        return {"error": str(e)}

@app.get("/api/knowledge")
async def get_knowledge():
    """获取知识库列表和内容"""
    return {
        "files": list(knowledge_manager.knowledge_cache.keys()),
        "content": knowledge_manager.get_knowledge_content()
    }

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

**关键点：**
- RESTful API 设计
- 会话管理
- 错误处理

## 七、预设场景

### 场景1：产品连接问题
**客户特征：** 新手用户，不太懂技术
**对话流程：**
1. 客户：我的手柄连不上蓝牙怎么办？
2. 客服：（需要给出正确的排查步骤）
3. 客户：还是连不上，还有其他方法吗？
4. 客服：（继续指导）

### 场景2：退换货咨询
**客户特征：** 比较着急
**对话流程：**
1. 客户：我买的手柄能退吗？
2. 客服：（说明退换货政策）
3. 客户：那怎么操作呢？
4. 客服：（告知流程）

### 场景3：功能咨询
**客户特征：** 想了解特定功能
**对话流程：**
1. 客户：这个手柄有连发功能吗？
2. 客服：（介绍功能）
3. 客户：怎么设置呢？
4. 客服：（说明设置步骤）
